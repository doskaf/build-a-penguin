# Build-A-Penguin

A very simple Sinatra application where you can create your own penguins!


## Installation

Clone this repository and run:

    $ bundle install

And then execute:

    $ rake db:migrate

Finally, run

    $ shotgun

You should be able to see the application at `localhost:9393`
